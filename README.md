# Proyecto Claudio Hornig - Sistema de Sanación Emocional

Este proyecto es una plataforma de aterrizaje y embudo de ventas para los servicios de Claudio Hornig.

## Estructura de Archivos (Orden Lógico)

1. **index.html**: Página de inicio y landing page principal. Punto de entrada para los usuarios.
2. **masterclass.html**: Página de registro para la clase magistral gratuita "Sanación Emocional Profunda".
3. **curso.html**: Página de ventas enfocada en el curso "El Camino de la Sanación".
4. **checkout.html**: Pasarela de pago simplificada para finalizar la compra del curso.
5. **agenda.html**: Sistema de reserva para sesiones privadas 1-a-1.

## Documentación
- **DESIGN.md**: Guía de estilo, paleta de colores (Primary: #003857, Secondary: #39665f) y principios de diseño editorial.

## Flujo del Embudo
Inicio -> Registro Masterclass -> Oferta de Curso -> Finalizar Compra (Checkout)
